# Molder.Generation
------
## Шаги для генерации тестовых данных

------
>  Сохранение даты в переменную.
```c#
[StepDefinition(@"я сохраняю дату ([0-9]{1,2})\.([0-9]{2})\.([0-9]+) в переменную ""(.+)""")]
```
>> Example
```gherkin
Given я сохраняю дату 1.5.2000 в переменную "test"
```
